Get Your Ubuntu to Look like Mac OS X

  • Misc
  • 2019-05-14

ubuntu2mac — Ubuntu TLS 18.04美化Mac OS风格。

最终效果图：

0x00 修改Gnome主题

https://www.gnome-look.org/p/1275087/

安装Tweak

  • sudo apt-get install gnome-tweak-tool

安装chrome-gnome-shell

  • sudo apt-get install chrome-gnome-shell

用浏览器打开地址https://extensions.gnome.org ，点击页面的Click here to install browser extension按钮进行扩展的安装。

  • 点击页面extensions，启用User Themes扩展。
  • 下载并启动blyr扩展：用于调整状态栏的透明度，Win键背景透明度。
    • 解决点击设置时报错：sudo apt install gir1.2-gtkclutter-1.0
  • 下载并启动netspeed扩展：任务栏显示网络速度。
  • 下载并启动coverflow alt-tab：切换窗口(Win + Tab)的主题。
  • 下载并启动hide dash x：隐藏(Win键)搜索时左边出现的Dock。

到站点https://www.gnome-look.org 下载Gnome Theme、icons、鼠标指针、GDM、Plymouth、字体。

  • Gnome Theme：MacOS UI主题。
  • Gnome-shell：任务栏、Terminal的主题。（按照gnome-shell.css文件渲染。）
  • icons：应用程序图标。
  • GDM：锁屏界面的主题。https://www.gnome-look.org/p/1207015/
  • Plymouth：开机启动动画。

  • 将下载的主题文件（Gnome主题、Gnome-shell主题）解压后放到用户家目录下的.themes目录下：~/.themes。

  • 将下载的图标文件(icons)和鼠标指针主题文件解压后放到用户家目录下的.icons目录下：~/.icons。

  • 把字体文件拷贝到~/.local/share/fonts目录下。

    • 注：.themes和.icons需要自己创建。Ctrl+H可以显示目录下的隐藏文件。

之后，打开tweak选择主题、游标、图标、字体。。。。

设置窗口的按钮（最小化、最大化、关闭）在左边：

  • Ubuntu：打开Tweak——Windows——Tilebar buttons——Placement——left
  • Kali：打开Tweak——Windows Title——Tilebar buttons——Placement——left

壁纸网站（需科学上网）：https://wallpapersite.com/

0x01 Dock

Dock方面使用plank，并把默认的Dock卸载掉。

卸载Ubuntu自带的Dock:

  • sudo apt remove gnome-shell-extension-ubuntu-dock

安装Plank dock：

  • sudo apt-get install plank

设置Plank开机启动：

  • Win键打开搜索 ==> 搜索startup ==> 打开startup application ==> Add ==> command处输入plank即可。

更改Plank主题：

  • 主题下载地址：https://www.gnome-look.org/p/1201720/
  • 将下载的主题文件放到：~/.local/share/plank/theme/目录下。
  • 在plank的preference里选用主题即可。

0x02 Zsh

  • 查看系统已安装的shell有哪些：cat /etc/shells
  • 查看系统当前shells：echo $SHELL

1.安装zsh：

  • apt-get install zsh
  • 为user用户设置 zsh 为系统默认 shell：chsh -s /bin/zsh user
  • 如果你要重新恢复到 bash：chsh -s /bin/bash user

2.安装zsh配置框架oh-my-zsh：

  • wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O - | sh
  • Oh-My-Zsh 的安装目录：~/.oh-my-zsh
  • Zsh 的配置文件位置：~/.zshrc

3.Zsh主题配置：

主题文件所在目录为~/.oh-my-zsh/themes。

  • ls ~/.oh-my-zsh/themes | wc -l
  • 主题介绍：https://github.com/robbyrussell/oh-my-zsh/wiki/Themes

修改主题：vim ~/.zshrc

  • 找到：ZSH_THEME="robbyrossell"改为自己喜欢的，默认是robbyrossell，想随机就使用random。
  • 自用不错的主题：fino、agnoster
    • agnoster会出现箭头乱码，是因为没有安装powerline字体：https://github.com/powerline/fonts

4.Zsh插件配置：

插件文件所在目录为~/.oh-my-zsh/plugins

  • ls ~/.oh-my-zsh/plugins | wc -l
  • 插件介绍：https://github.com/robbyrussell/oh-my-zsh/wiki/Plugins

启用插件：vim ~/.zshrc

  • 找到plugins=(git)，括号（）中即要要启用的插件，默认启用了git，多个用空格分开。
  • 修改启用插件后让配置文件生效：source ~/.zshrc

对于Zsh没有自带的插件，可以将其下载放到：~/.oh-my-zsh/custom/plugins目录下，然后再在配置文件中启用，启用方法同上。

安装两个常用的外部插件：

#安装外部插件：zsh-syntax-highlighting，作用是zsh语法高亮。
cd ~/.oh-my-zsh/custom/plugins
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git

#安装zsh-autosuggestions
cd ~/.oh-my-zsh/custom/plugins
git clone https://github.com/zsh-users/zsh-autosuggestions.git

下载完毕后，到配置文件启用即可。
